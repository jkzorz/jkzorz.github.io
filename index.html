Analyzing Microbial Ecology Data

I'm using these tutorials as a place to keep track of code I've used to generate R visualizations and statistical analyses, specifically for the purpose of visualizing and analyzing large microbial ecology datasets. I also talk about the analyses/visualizations here in terms of microbial amplicon/marker gene data, but these analyses can be extended for use in nearly any situation with a sample x species abundance/frequency, or sample x numerical variable type output.

Disclaimer: I am in no way an R expert, which is perhaps evident in my code. However, all the code in my tutorials has worked for me in the past, and gotten me the results I wanted. There are probably hundreds of different and/or more elegant ways to obtain the same end product in R, so feel free to adapt things as needed.

I'm also not a statistics expert by any means, and I rely heavily on some really great resources out there. My go to is GUSTAME, which is a guide to statistical analysis in microbial ecology. Also their papers:

  • Multivariate analyses in microbial ecology
  • A guide to statistical analysis in microbial ecology: a community-focused, living review of multivariate data analyses
